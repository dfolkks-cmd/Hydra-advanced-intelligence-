# HYDRA Raw Mode AI Personality

## Deployment on iPhone

1. Install Pythonista or Pyto.
2. Copy the folder `HYDRA_Raw_Mode` to the app's script directory.
3. Open `hydra_raw_tts.py`.
4. Run the script to test voice output.
5. Optional: Create a Shortcut:
   - "Run Script" → select `hydra_raw_tts.py`
   - Add voice or button triggers
6. Enjoy HYDRA Raw Mode energy, motivational and direct.

⚡ Notes:
- Prosody settings simulate intensity
- Categories: greeting, motivational, status, fallback
- Fully safe, streetwise, and commanding style
